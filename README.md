An AI-powered Machine Learning web application that predicts whether a patient is likely to have **Malaria** or **Typhoid** based on symptoms.

## 📌 Project Overview

This project was developed using Machine Learning to classify common infectious diseases from patient symptoms.

The model analyzes medical information provided by the user and predicts whether the patient is more likely to have:

- 🦟 Malaria
- 🤒 Typhoid

The application also displays the prediction confidence for each disease.
